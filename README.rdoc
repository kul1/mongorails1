== Rails 4 APP using MongoDB(Mongoid). 
App has posts,comments and admin panel. Comments are embedded into posts. Admin panel uses rails_admin mounted at /admin. Admin is protected with cancan that allows users with admin=true to use the panel.
Uses http://requiremind.com/riding-rails-4-along-with-mongoid-and-ruby-2-dot-0/ as a reference and starting point.
